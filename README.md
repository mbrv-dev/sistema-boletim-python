# 📊 Student Grade System (Sistema de Boletim em Python)

Um mini sistema desenvolvido em **Python** para cadastrar alunos, registrar notas e calcular médias automaticamente.

## 🚀 Funcionalidades

* Cadastro de alunos
* Registro de 3 notas por aluno
* Cálculo automático da média
* Classificação da situação do aluno:

  * ✅ Aprovado
  * ⚠️ Recuperação
  * ❌ Reprovado
* Identificação da **maior média** da turma
* Identificação da **menor média** da turma

## 🧠 Conceitos de programação utilizados

Este projeto foi criado para praticar conceitos importantes de programação:

* Listas
* Loops (`for`)
* Estruturas condicionais (`if / elif / else`)
* Tratamento de erros (`try / except`)
* Funções nativas do Python (`sum`, `len`, `max`, `min`)
* Função `lambda`

## 💻 Tecnologias

* Python 3

## 📷 Exemplo de execução

```
Aluno: Ana
Notas: [8.0, 9.0, 7.0]
Média: 8.00
Situação: Aprovado!
```
